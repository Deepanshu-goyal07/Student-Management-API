# Student-Management-API
Student Management API is a beginner-level backend project built using Node.js and Express.js. It provides CRUD operations to add, view, update, and delete student records. Student data is stored in a local JSON file using the fs module, eliminating the need for a database. The project also includes routing, middleware, RESTful APIs, and simple HTML forms for user interaction, making it a practical introduction to backend development concepts.
